    Demo scenes

    AWSIM includes the following demo scenes.

    Scene Feature
    AutowareSimulationDemo.unity This is the main demo simulation scene connecting AWSIM and Autoware for self-driving.
    IntegrateScenarioSimulatorDemo.unity Demo simulation connecting AWSIM and Scenario simulator v2 to run the sample Open SCENARIO.
    PcdGenerationDemo.unity Demo simulation outputting pcd from 3D model of AWSIM map
